# Security Policy

## Supported Versions
| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |

## Reporting a Vulnerability
ANFA Layer is a security product. We take vulnerabilities seriously.

Please report vulnerabilities via GitHub Security Advisories:
https://github.com/Mpitafi7/ANFA-LAYER/security/advisories/new

Do NOT report security issues in public GitHub issues.

We will respond within 48 hours and provide a fix within 7 days
for confirmed vulnerabilities.
